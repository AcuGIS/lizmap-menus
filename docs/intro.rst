Intro
===========================

Overview
------------

pg_layers is a stand-alone PHP application for viewing and sharing PostGIS data via Leafletjs.

It supports Users, Groups, and map-level permissions.


.. image:: _static/JRIViewer-Main.png


System Requirements
-------------------
* PostgreSQL
* PHP >= 8.0
